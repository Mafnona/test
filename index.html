1. Основы HTML

HTML (HyperText Markup Language) - это основа веб-страницы. Он определяет структуру контента.

Базовая структура HTML-документа:

<!DOCTYPE html>
<html>
<head>
  <title>Заголовок страницы</title>
</head>
<body>
  <!-- Содержимое страницы -->
</body>
</html>

Основные теги:

  • <h1>-<h6> - заголовки
  • <p> - параграф
  • <a> - ссылка
  • <img> - изображение
  • <div> - блок
  • <span> - строчный элемент

Попробуй сам:

Создай простую HTML-страницу с заголовком, параграфом и ссылкой.

<h1>Мой первый заголовок</h1>
<p>Это мой первый параграф на HTML-странице.</p>
<a href="#">Это ссылка</a>

2. Основы CSS

CSS (Cascading Style Sheets) используется для стилизации HTML-элементов.

Способы добавления CSS:

  • Inline (в строке) - с помощью атрибута style
  • Внутренний - с помощью тега <style> в <head>
  • Внешний - с помощью отдельного .css файла

Базовый синтаксис CSS:

селектор {
  свойство: значение;
}

Основные свойства CSS:

  • color - цвет текста
  • background-color - цвет фона
  • font-size - размер шрифта
  • margin - внешние отступы
  • padding - внутренние отступы

Попробуй сам:

Создай стиль для заголовка h1: красный цвет текста, размер 32px, выравнивание по центру.

h1 {
  color: red;
  font-size: 32px;
  text-align: center;
}

3. Макеты и позиционирование

Современная верстка использует Flexbox и Grid для создания макетов.

Flexbox:

Идеален для одномерных макетов (в строку или колонку).

.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

CSS Grid:

Идеален для двумерных макетов (и строки, и колонки).

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 20px;
}

Попробуй сам:

Создай контейнер с тремя блоками, расположенными в строку с равным распределением пространства.

.container {
  display: flex;
  justify-content: space-between;
}

.block {
  width: 30%;
  background: #eee;
  padding: 20px;
}

4. Адаптивная верстка

Адаптивная верстка ensures ваш сайт хорошо выглядит на всех устройствах.

Viewport meta tag:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Медиа-запросы:

@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }
}

Попробуй сам:

Создай медиа-запрос, который меняет цвет фона на светло-серый на экранах меньше 600px.

@media (max-width: 600px) {
  body {
    background-color: #f0f0f0;
  }
}

5. Практика верстки

Лучший способ научиться верстать - практиковаться! Вот несколько идей для проектов:

  • Создай личную страницу-визитку
  • Сверстай страницу с фотогалереей
  • Создай блог с несколькими статьями
  • Сделай страницу с контактной формой
  • Попробуй повторить layout известного сайта

Советы для практики:

  • Начинай с простых проектов и постепенно усложняй
  • Используй валидатор HTML/CSS для проверки кода
  • Проверяй верстку на разных устройствах и браузерах
  • Изучай код других разработчиков
